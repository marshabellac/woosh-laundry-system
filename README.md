## Deskripsi Proyek
Proyek ini dibuat untuk memenuhi tugas mata kuliah Object Oriented Programming (OOP). Sistem yang dibangun adalah sistem manajemen laundry untuk membantu pengelolaan data pelanggan, transaksi, status cucian, dan pembayaran.

## Tujuan Proyek
- Menerapkan konsep Object Oriented Programming (OOP)
- Membuat sistem yang dapat membantu operasional laundry
- Mengelola data pelanggan dan transaksi dengan lebih rapi
- Menyediakan alur kerja laundry yang lebih terorganisir melalui antarmuka GUI

## Fitur yang Direncanakan
- Login admin
- Manajemen data pelanggan
- Manajemen paket laundry
- Input transaksi laundry
- Perhitungan harga otomatis
- Pembayaran
- Update status cucian
- Pengambilan laundry
- Pencatatan biaya operasional

## Teknologi yang Digunakan
- **Bahasa Pemrograman:** C#
- **Interface:** GUI
- **Version Control:** GitHub
